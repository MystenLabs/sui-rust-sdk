Authenticated event stream verification.

AuthenticatedEventsClient drives the v2 LedgerService.ListEvents RPC filtered by EventStreamHeadFilter, replays each received event into a local Merkle Mountain Range, and periodically reconciles that MMR against the on-chain EventStreamHead fetched via an OCS inclusion proof. Events are only yielded to the consumer after the reconciliation that covers them succeeds, so every emitted event is cryptographically authenticated end-to-end.

AuthenticatedEvent is the in-memory envelope used both as the decoded form of a ListEventsResponse item and as the unit of work flowing through the verifier.
